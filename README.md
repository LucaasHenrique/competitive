## 📌 About

Welcome to my centralized **Competitive Programming** and **Programming Marathons** repository.
This space is dedicated to my studies, problem solutions across various online judges, theoretical materials, classic books, as well as algorithm and data structure implementations.

---

## 📁 Repository Structure

The repository is divided into the following main folders:

- 📚 **[`Biblioteca/`](./Biblioteca/)**
  - Collection of pre-implemented and optimized algorithms and data structures, used as base *templates* for programming marathons.

- 📖 **[`competitive-material/`](./competitive-material/)**
  - Repository of essential theoretical materials and books for CP (*Competitive Programming 3 and 4*, *Introduction to Algorithms (Cormen)*, *CP Handbook*, *PAPS*, among others).

- 💻 **[`competitive-programming/`](./competitive-programming/)**
  - The coding core. Contains solutions for various problems, categorized by platforms and competitions:
    - **CP Judges:** `Codeforces`, `Beecrowd/URI`, `CSES`, `AtCoder`, `Kattis`, `USACO`
    - **Competitions:** `OBI` (Brazilian Olympiad in Informatics), `SBC Marathon`, `Gaúcha Marathon`, university lists (`USP`, `UFMG`), etc.

- 🏗️ **[`ds-c/`](./ds-c/)**
  - Didactic and detailed implementations of Classic Data Structures written purely in **C** (*Heaps, Lists, Stacks*).
